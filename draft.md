我不希望我的每個常用工具都散落在不同地方，因此我提出以下構想:

利用 docker 製作的拋棄式跨平台工具集，從簡單的檔案整理、圖像處理到影像轉換

永遠的入口:
```powershell
docker run --rm -v ${PWD}:/data sys-toolkit
```

目前先提供:
- 統計行數
- 檢查空資料夾
- 快速解壓縮
- 圖像轉換 (將直接子圖像檔案全部轉為指定格式並刪除原本的(可選，預設刪除))
- 影像擷取 (在之前的功能上新增輸出到單獨目錄或統一目錄並命名成 統一前墜 + 0001.jpg 的功能)
- yt-dlp 包裝 (只用來下載公開資源，不支援需登入的情況)

README 分幾個章節，第一個是功能概覽，表格型，兩欄位，標題與描述(用途，比如 檢查空資料夾，git 切換分支後，有時會莫名留下空資料夾，可用該工具檢查)

第二個是安裝與使用(啟動)方式

第三個與後續是每個功能的使用方式

基礎使用 Debian Slim ，雖然大一些，但應該更容易安裝依賴與寫腳本
